
这里包含两种Kmeans算法
第一种是最基础的Kmeans，其迭代终止条件为样本聚类不再发生改变
第二种是基于基础Kmeans的二分Kmeans算法，其循环终止条件为码本数量达到所设定的数量，其基本原理是不断使用基础的Kmeans进行二分类，而后选取误差值改变最大的种群进行第二次分类，直至码本数量达到预设数量

1. 基础kmeans:
# K均值聚类算法
# 输入：训练集dataMat，码本数量k
# 输出：簇中心centroids, 各个样本分类clusterassment
算法流程：
    初始化k个点作为起始质心
    当样本中的任一一个样本发生改变时，进入循环：
        对每一个样本：
            对每一个质心：
                计算样本与质心距离
                求出样本与质心的最小距离
                记录该最小距离的质心编号
            更新样本质心所属
        对每一个簇，取簇中样本平均值作为更新质心
    返回簇中心坐标和各个样本分类

2. 二分Kmeans算法：
# 二分K均值聚类算法
# 输入： 样本集dataMat,码本数量k
# 输出： 簇中心centroids, 各个样本分类clusterassment
算法流程：
    将所有点看为一个簇
    当簇数目小于k时：
        对每一个簇：
            计算分类前总误差
            调用kmeas进行二分类，计算分类后总误差
        选择分类后总误差最小的簇进行二分类
